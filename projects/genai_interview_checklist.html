Technical Decisions — Be Ready to Defend
Why LangGraph over plain LangChain?
Stateful, conditional edges, auditability, each agent returns only changed fields
MUSTLOANIQ
Why hybrid retrieval (Dense + BM25 + RRF)?
Dense catches meaning, BM25 catches exact terms like "DTI 43%". RRF merges fairly
MUSTLOANIQ
Why different models for different agents?
Cost optimization: Llama free, Haiku cheap for PII, Sonnet for complex decisions
MUSTLOANIQ
Why AWS Bedrock for Compliance + Underwriting + Decision agents?
PII data never leaves AWS VPC — regulatory compliance requirement
MUSTLOANIQ
Why pgvector instead of Pinecone?
Single DB for vectors + metadata + app data. Simpler ops, metadata filtering built-in
MUSTLOANIQ
Why fine-tune Llama 3.1 for RatioAgent?
$0 marginal cost per call vs $0.005 GPT-4o. Narrative quality 87% of GPT-4o
GOODLOANIQ
How do you handle LLM failures gracefully?
Fallback defaults in every agent — pipeline never crashes, errors propagate in state
